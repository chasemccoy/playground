import Text from 'components/Text'

# The `Text` Component

```.jsx
<Text.p>This is a paragraph.</Text.p>
```

The `Text` component renders a `div` element and offers the following `styled-system` utilities:

- `space`
- `color`
- `fontSize`
- `fontWeight`
- `lineHeight`

And the following sub-components:

- `Text.p`

```.jsx
<Text.p color='blue.600' fontSize={5}>This text is blue.</Text.p>
```
