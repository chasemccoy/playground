import Heading from 'components/Heading'

# The `Heading` Component

```.jsx
<Heading>This is a heading.</Heading>
```

The `Heading` component renders heading elements and offers all of the `styled-system` utilities as [the `Text` component](text), as well as the following sub-components:

- `Heading.h1`
- `Heading.h2`
- `Heading.h3`
- `Heading.h4`
- `Heading.h5`
- `Heading.h6`

```.jsx
<Heading.h2 color='green.600'>This text is green.</Heading.h2>
```
